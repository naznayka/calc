У середовищі Visual Studio Code було створено інтерактивний калькулятор на мові HTML з використанням JavaScript.
Калькулятор виконує основні арифметичні операції: додавання, віднімання, множення та ділення.
Користувач вводить два числа та обирає потрібну операцію за допомогою кнопок.
Результат обчислення відображається на сторінці, також реалізовано перевірку ділення на нуль.
Також реалізовано кнопку очищення всіх полів введення та результату.
